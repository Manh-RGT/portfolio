# Tran Van Manh — Portfolio Website

Personal portfolio site for **Tran Van Manh**, a Robotics Engineer specializing in deep reinforcement learning, autonomous navigation, and embedded systems.

## Tech Stack

- Plain HTML / CSS / JavaScript — no build tools required
- [Font Awesome](https://fontawesome.com/) for icons
- [Google Fonts](https://fonts.google.com/) (DM Sans + Space Mono)

## Project Structure

```
portfolio/
├── index.html       # Main page (all sections)
├── style.css        # All styles
├── script.js        # Scroll animations, interactions
├── image/           # Profile and project images
├── CV/              # Resume PDF
└── project01/       # Project detail pages
```

## Website from github
https://manh-rgt.github.io/portfolio/

## Edit Content

| What to change | Where |
|---|---|
| Personal info, bio, skills | `index.html` — `#about` section |
| Work experience | `index.html` — `#experience` section |
| Projects | `index.html` — `#projects` section |
| Publications | `index.html` — `#publications` section |
| Colors / fonts / layout | `style.css` |
| Animations / scroll behavior | `script.js` |

## Deploy

Drop the entire folder into any static hosting service:

- **GitHub Pages** — push to a repo, enable Pages in Settings
- **Netlify / Vercel** — drag-and-drop the folder in the dashboard
